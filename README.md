## Эндпоинты

### `api/RefreshStocks`

Загрузка из API в БД данных из раздела **«Склады»**.  
**Таблица в БД:** `stocks`

**Параметры:**
- `key` (обязательный) — API-ключ.


### `api/RefreshIncomes`

Загрузка из API в БД данных из раздела **«Доходы»**.  
**Таблица в БД:** `incomes`

**Параметры:**
- `key` (обязательный) — API-ключ.
- `dateFrom` (обязательный) — дата начала периода (формат `Y-m-d`).
- `dateTo` (обязательный) — дата окончания периода (формат `Y-m-d`).
